Cтелс-шутер 2

1. Название проекта: «Стелс-шутер 2»
<img width="661" height="424" alt="image" src="https://github.com/user-attachments/assets/8a8c800a-a2ab-4d01-8e6e-cf9a789eee49" />

2. Жанр: Third-Person Shooter
3. Платформа: ПК
4. Управление: WASD, R, Q, C, Space, Shift, мышь
5. Цель игры: Пройти все уровни.
6. В игре присутствует возможность настроить качество графики, громкость звука.
<img width="792" height="506" alt="image" src="https://github.com/user-attachments/assets/0782d956-d8d3-481a-a4ac-de50e691daf1" />

Элементы геймплея

1. Управление персонажем:
Игрок управляет персонажем с помощью клавиш WASD для передвижения по уровню
Игрок управляет камерой с помощью мыши для обзора окружения
Игрок может бегать, приседать, перезаряжаться, стрелять, прицеливаться.
<img width="792" height="506" alt="image" src="https://github.com/user-attachments/assets/19a47a3d-a0c6-4613-9c15-f1ccf5545926" />

2. Система здоровья: У персонажа есть здоровье и жизни, если здоровье кончится, отнимется жизнь и уровень начнётся заново, если жизни кончатся, игра проиграна - нужно начинать заново.
3. Враги-боты: Главное препятствие - боты, они патрулируют свою территорию, могут видеть и слышать игрока, если вас обнаружат будут стрелять.
<img width="792" height="506" alt="image" src="https://github.com/user-attachments/assets/a2e82845-c960-44d7-b7ab-f5fc3bc083f8" />

4. Квесты и задачи: В игре 3 уровня и 4 квеста, чтобы пройти квесты нужно выполнять задачи квеста (убить ботов, не дав им убить персонажа, не дать ботам себя обнаружить).

5. Логика предметов: На уровнях есть 2 вида предмета:
Патроны - нужны чтобы убить ботов.
Аптечка - восстанавливает здоровье.
